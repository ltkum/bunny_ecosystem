# Commands

This part of the bot will contain the following:

1. function with commands
2. groups of commands
3. commands registration

a directory contains a group of command, and has the following files

1. one file per command
2. one file for the group, which adds the command

there is a `base` directory which contains no group

at the root of this directory, there is the Bunny group, which contains all others, and the permission utility.
